Forms

Form control styles, layout options, and custom components.

Bootstrap docs

Range slider depends on noUISlider plugin. Make sure to link to noUISlider css and js files in your document: assets/vendor/nouislider/dist/nouislider.min.css and assets/vendor/nouislider/dist/nouislider.min.js. Use this page as a reference.

To enable input text content formatting please make sure to link to Cleave.js plugin in your document: assets/vendor/cleave.js/dist/cleave.min.js. Use this page as a reference.
