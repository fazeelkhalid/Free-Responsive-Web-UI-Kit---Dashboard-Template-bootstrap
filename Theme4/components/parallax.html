Parallax

Silicon supports 2 types of parallax effect: on page scrolling and cursor position.

Silicon component

In your document make sure to link to:

  • assets/vendor/jarallax/dist/jarallax.min.js - for background image parallax
  • assets/vendor/rellax/rellax.min.js - for element parallax
  • assets/vendor/parallax-js/dist/parallax.min.js - for cursor position parallax
  • assets/vendor/vanilla-tilt/dist/vanilla-tilt.min.js - for 3D tilt effect

Use this page as a reference.
