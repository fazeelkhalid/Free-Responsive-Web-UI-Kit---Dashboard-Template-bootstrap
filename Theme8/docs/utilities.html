Utility classes

Bootstrap includes dozens of utilities - classes with a single purpose to reduce the frequency of highly repetitive declarations.

Along with default Bootstrap utility classes Cartzilla introduces its own. They all live inside Cartzilla/src/scss/theme/_utilities.scss file.

The list of most important Cartzilla utilities as well as their visual reference and code snippets can be found here.
